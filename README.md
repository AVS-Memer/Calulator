# Calulator
You input 2 numbers, and an operator, and it will calculate the answer

# 2 Number Operators:
# +
Adds 2 numbers
# -
Subtracts 2 numbers
# *
Multiplies 2 numbers
# /
Divides 2 numbers
# %
Finds one number percent of the other number
# ^
Raises a number to the power of another number
# R
Divides 2 numbers with a remainder

# 1 Number Operations:
# ++
Adds 1 to a number
# --
Subtracts 1 from a number
# %
Finds the percentage of a number
# *2
Doubles a number
# *3
Triples a number
# /2
Finds the half of a number
# ^2
Squares a number
# ^3
Cubes a number
# \/
Gets the floor of a number; rounds it down
# /\
Gets the ceiling of a number; rounds it up
